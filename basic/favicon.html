Use the HTML element to insert a favicon

A favicon image is Display to the left of the page title in the browser tab
To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder. A common name for a favicon image is "favicon.ico". Next, add a element to your "index.html" file, after the element, like this: </p> </body> </html>
